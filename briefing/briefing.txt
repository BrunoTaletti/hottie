Brefing Cliente Sistema "Hottie"
Data: 26/06/2023

Problema: O Cliente possui problemas para realizar a gestão do seu negócio de nicho hot, muita demora em cadastrar pedidos e visualizar de qual funcionário veio cada venda além da folha de ponto.

Desejo: Realizar a construção de um software web para a gestão do seu negócio digital voltado ao nicho hot.

Necessidade: Criar um software que seja possivel ser acessado via navegador e de forma remota para os funcionários, ao qual seja possivel realizar o controle de ponto dos funcionários e as vendas feitas por cada um.

O que será feito?

Será realizado um sistema simples em HTML, CSS e PHP com banco de dados MYSQL ao qual será um CRUD para efetuar manipulação de dados no banco de forma visual.

Primeiramente iremos desenvolver o design do wireframe com o cliente para que seja possivel acertar todo o escopo, visual e funcionalidades necessárias.

Por ultimo iremos desenvolver o design do sistema como deverá parecer em seu MVP final próximo a entrega do projeto.
